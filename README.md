# SQLx formatter

[![Rust](https://github.com/Lindronics/sqlx-fmt/actions/workflows/rust.yml/badge.svg)](https://github.com/Lindronics/sqlx-fmt/actions/workflows/rust.yml)

Finds als SQLx `query!` and `query_as!` macros in a file and formats them using an inner formatter.

## Supported formatters

- [`pg_format`](https://github.com/dalibo/pg_format)
